# NatiCure - Smart Remedy Suggestion System

## Overview
NatiCure is a Django-based web application that suggests remedies based on user symptoms with multilingual support.

## Features
- Symptom-based search
- Remedy suggestions
- Telugu and English support
- Dynamic UI

## Tech Stack
- Python
- Django
- HTML, CSS
- SQLite

## Setup
pip install -r requirements.txt  
python manage.py runserver
